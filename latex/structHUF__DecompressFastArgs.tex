\doxysubsection{Detailed Description}
The input/output arguments to the Huffman fast decoding loop\+:

ip \mbox{[}in/out\mbox{]} -\/ The input pointers, must be updated to reflect what is consumed. op \mbox{[}in/out\mbox{]} -\/ The output pointers, must be updated to reflect what is written. bits \mbox{[}in/out\mbox{]} -\/ The bitstream containers, must be updated to reflect the current state. dt \mbox{[}in\mbox{]} -\/ The decoding table. ilowest \mbox{[}in\mbox{]} -\/ The beginning of the valid range of the input. Decoders may read down to this pointer. It may be below iend\mbox{[}0\mbox{]}. oend \mbox{[}in\mbox{]} -\/ The end of the output stream. op\mbox{[}3\mbox{]} must not cross oend. iend \mbox{[}in\mbox{]} -\/ The end of each input stream. ip\mbox{[}i\mbox{]} may cross iend\mbox{[}i\mbox{]}, as long as it is above ilowest, but that indicates corruption.
