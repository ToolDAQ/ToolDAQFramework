\doxysubsection{Detailed Description}
This class handels the service discovery for the Tool\+Chain. It oth handels publishing multicast beacons for the Tool\+Cahin and any other customs services and it collates and maintains a list of other remote services form their multicast beacons that Tools can acess via a ZMQ port and query.
