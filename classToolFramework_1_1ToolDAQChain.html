Parameters
verboseThe verbosity level of the ToolDAQChain. The higher the number the more explicit the print outs. 0 = silent apart from errors.
errorlevelThe behavior that occurs when the ToolDAQChain encounters an error. 0 = do not exit for both handeled and unhandeled errors, 1 = exit on unhandeled errors only, 2 = exit on handeled and unhandeled errors.
serviceThe name of the ToolDAQChain service to boradcast for service discovery
logmodeWhere log printouts should be forwarded too. "Interactive" = cout, "Remote" = send to a remote logging system, "local" = ouput logs to a file.

log_addressService name of remote logging system if in Remote logging modemulticast address for remote logging.
log_portThe Port number of the remote logging service
pub_secThe number of seconds between publishing multicast serivce beacons
kick_secThe number of seconds to wait before removing a servic from the remote services list if no beacon received.
IO_ThreadsThe number of ZMQ IO threads to use (~1 per Gbps of traffic).
